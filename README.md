# Forzapp

ERP Modular - Sistema de gestión corporativo con módulos administrativos, kioskos/terminales locales y sesiones móviles
para conductores, integradas con impresoras térmicas Bluetooth.

  progreso.

## Instalación y ejecución con Bun

1. Instala dependencias:
   ```sh
   bun install
   ```

2. Ejecuta el entorno de desarrollo:
   ```sh
   bun run dev
   ```
